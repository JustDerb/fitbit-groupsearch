Stats for Nerds

Below are various metrics about this site. This is to let you know how the sites health is, and also provide some fun facts from pulling data from FitBit. This data is ever-changing, so come back every so often to see updates!

Indexed Groups (Month)

The overall number of groups that have been indexed so far, each month. Expect this number to just keep going up! (Updated every hour)

Group Runs (Sum)

Each hour, how many full scans have been run? (Updated every hour)

Group Runs Successful

Status of runs by the hour. 1 = fully scanned, 0 = error. If there are any 0 datapoints, an alarm will fire for investigation. (Updated every hour)
